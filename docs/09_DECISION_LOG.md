# 09_DECISION_LOG.md

## Decision 1: Use GitHub Pinned Repos
Reason:
Projects should stay synchronized with GitHub. The portfolio should update when pinned repositories change.

## Decision 2: Use Backend API Route for GitHub
Reason:
GitHub token must not be exposed in frontend.

## Decision 3: Do Not Auto-Request Location
Reason:
Asking location on page load feels intrusive. The user must click a button first.

## Decision 4: No Database in MVP
Reason:
The MVP does not need persistent storage.

## Decision 5: Minimal Dark UI
Reason:
The website should look clean, premium, and professional.
