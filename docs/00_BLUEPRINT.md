# 00_BLUEPRINT.md

## Product Name
Hardik Singh Portfolio

## Product Type
Minimal developer portfolio website.

## Main Idea
A premium dark-themed portfolio that automatically syncs pinned GitHub repositories, displays coding activity from GitHub and LeetCode, shows skills, hackathons, resume, and a privacy-friendly contact section.

## Target Users
- Recruiters
- Hackathon judges
- Developers
- College peers
- Hiring managers

## Main Value
This portfolio should prove that Hardik can:
- Build polished frontend interfaces.
- Integrate real APIs.
- Work with GitHub GraphQL.
- Handle errors properly.
- Create clean UI/UX.
- Think like a product developer.

## Core Features
1. Minimal hero section.
2. GitHub-synced Projects section.
3. GitHub and LeetCode coding activity.
4. Skills section.
5. Hackathons section.
6. Resume download.
7. Contact form.
8. Optional viewer-location map.

## MVP Scope
Build only what is necessary for the first version.

Included:
- Single-page portfolio.
- GitHub pinned projects sync.
- Coding stats cards.
- Resume button.
- Contact section.
- Responsive layout.

Not included:
- Blog.
- Admin dashboard.
- Login system.
- Database.
- Heavy animations.
- 3D effects.
