# 04_UI_UX_GUIDE.md

## Visual Direction
The website should look:
- Minimal
- Dark
- Premium
- Technical
- Calm
- Recruiter-friendly

## Colors
```css
--background: #030303;
--card: #070707;
--card-hover: #0a0a0a;
--border: #242424;
--accent-border: #7c6dff;
--text-main: #ffffff;
--text-secondary: #b5b5c3;
--text-muted: #71717a;
--accent: #8b7cff;
```

## Layout Rules
- Max width: 1120px or 1200px.
- Section padding: `py-24` or `py-28`.
- Cards: `rounded-2xl` or `rounded-3xl`.
- Use large spacing.
- Avoid clutter.

## Card Style
Cards should have:
- Black background.
- Thin border.
- White title.
- Gray description.
- Muted metadata.
- Purple/blue border on hover.

## Animations
Allowed:
- Fade in.
- Small hover lift.
- Border glow.
- Smooth scroll.

Avoid:
- 3D effects.
- Particle background.
- Flashy gradients.
- Too many moving elements.
