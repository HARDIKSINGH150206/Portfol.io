# Hardik Singh Portfolio
A minimal dark developer portfolio that automatically syncs pinned GitHub repositories and displays coding activity from GitHub and LeetCode.

## Features
- Minimal dark UI
- GitHub pinned projects sync
- GitHub coding activity
- LeetCode stats
- Skills section
- Hackathons section
- Resume download
- Contact form
- Optional viewer-location map

## Tech Stack
- Next.js
- TypeScript
- Tailwind CSS
- GitHub GraphQL API
- LeetCode API
- React Leaflet
- Vercel

## Getting Started
Install dependencies:

```bash
npm install
```

Create `.env.local`:

```env
GITHUB_TOKEN=
GITHUB_USERNAME=
LEETCODE_USERNAME=
NEXT_PUBLIC_SITE_URL=
```

Run locally:

```bash
npm run dev
```

Build:

```bash
npm run build
```
